In this tutorial we walk through all the steps to creating a custom Elgg page. En este tutorial recorreremos todos los pasos para crear una página personalizada de Elgg. A custom page can be used to organize and manage any content. Se puede utilizar una página personalizada para organizar y administrar cualquier contenido. You can add static content like Company Information or pull content from the Elgg database. Puede agregar contenido estático como Información de la empresa o extraer contenido de la base de datos de Elgg.

We're going to create a page and populate it with a heading and three modules. Vamos a crear una página y llenarla con un encabezado y tres módulos. The modules will display a user avatar, friends and latest blog posts. Los módulos mostrarán un avatar de usuario, amigos y últimas entradas de blog.
